## 三、核心模块说明

| 模块 | 职责 | 文件 |
|------|------|------|
| Project Registry | 项目注册与管理 | lib/project-registry.js |
| Context System | 用户会话上下文 | lib/context-system.js |
| Task System | 任务生命周期管理 | lib/task-system.js |
| Memory System | AI记忆与历史 | lib/memory-system.js |
| State System | 状态同步管理 | lib/state-system.js |
| AI Worker | 调用AI生成代码 | workers/ai-worker.js |
| Git Worker | Git操作执行 | workers/git-worker.js |
| Jenkins Worker | Jenkins构建触发 | workers/jenkins-worker.js |
| Deploy Worker | SSH/K8s部署 | workers/deploy-worker.js |
| API Registry | API规则管理 | lib/api-registry.js |
| Notification | 通知推送 | lib/notification.js |
| Security | 安全确认机制 | lib/security.js |

## 四、数据流向

1. **输入流**: 微信/飞书/Web → API Gateway → 解析命令 → 获取上下文 → 创建任务
2. **执行流**: 任务调度 → Worker执行 → 状态更新 → 进度通知
3. **输出流**: 执行结果 → 通知系统 → 微信/飞书/Web响应
